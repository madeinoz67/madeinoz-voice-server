# madeinoz-voice-server

Welcome to the madeinoz-voice-server documentation.

## Overview

madeinoz-voice-server is a Text-to-Speech (TTS) service built with Bun, using the Qwen TTS model for high-quality speech synthesis.

## Features

- Fast TTS processing powered by Bun runtime
- Qwen TTS model integration via HuggingFace
- Simple REST API
- TypeScript for type safety
